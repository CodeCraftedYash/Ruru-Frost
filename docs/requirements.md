# Requirements

## Project Name

Ruru Frost

---

## Description

Ruru Frost is a single-brand e-commerce platform where customers can browse and purchase premium ice cream products.

The application also provides an admin dashboard for managing products, inventory, categories, and customer orders.

---

## Goals

- Build a production-like full-stack application
- Learn backend architecture
- Practice database design
- Learn Prisma ORM
- Learn authentication and authorization
- Improve API design
- Create a strong portfolio project

---

# User Roles

## Customer

Can:

- Register
- Login
- Browse products
- Search products
- Filter products
- Add products to cart
- Manage cart
- Place orders
- View order history
- Manage profile
- Manage addresses

---

## Admin

Can:

- Login
- Create products
- Update products
- Delete products
- Manage categories
- Upload product images
- Update stock
- View all orders
- Update order status
- View dashboard statistics

---

# Out of Scope

The following features will not be included in Version 1.

- Wishlist
- Coupons
- Reviews
- Email Verification
- Live Chat
- Multi-vendor Support
- Real Payment Gateway
